# praha-agile
疑似アジャイル開発課題用のリポジトリ

## セットアップ

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

http://localhost:3000

## 主要スクリプト

- `pnpm dev` / `pnpm build` / `pnpm start`
- `pnpm lint` / `pnpm lint:fix` / `pnpm format`
- `pnpm typecheck` / `pnpm test`
- `pnpm db:generate` / `db:migrate` / `db:push` / `db:studio`
